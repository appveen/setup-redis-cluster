# REDIS Cluster POC

## Steps to install redis cluster
 - Run the following command
   ```
    sudo ./install.sh
   ``` 
 - Specify the k8s namespace and cluster size as prompted

## Steps to uninstall redis cluster
 - Run the following command
   ```
    sudo ./uninstall.sh
   ```
 - Specify the k8s namespace from where you want to uninstall
